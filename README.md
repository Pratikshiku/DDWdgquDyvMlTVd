## Natural Environment Live Streaming System Based on RTSP/RTMP

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/7dc888dc159c4279ba9d2e26ab01acdf/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
System is divided into (roles): Regular users, Administrators.

**Regular User Features:**
1. Login
2. Registration
3. Live Room
4. My Live Room
5. My Follows
6. Browsing History
7. Watch Live Stream
8. Send Comments
9. My Favorites
10. Secure Exit


**Administrator Features:**
1. Login
2. Live Room Management
3. User Management
4. Browsing History Overview
5. Favorites Overview
6. Comment Management
7. Data Dashboard Console
8. Secure Exit
